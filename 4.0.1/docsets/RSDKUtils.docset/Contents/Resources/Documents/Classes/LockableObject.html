LockableObject

public class LockableObject<T> : LockableResource

Object-wrapper that conforms to LockableResource protocol. Used to control getter and setter of given resource. Can be used to create atomic transaction.

When lock() has been called on some thread, only that thread will be able to access the resource. Other threads will synchronously wait for unlock() call to continue.

Note

get() and set() are not thread-safe by itself
