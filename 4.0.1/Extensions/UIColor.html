UIColor

public extension UIColor
  • init(hexString:alpha:)

    Convert hexadecimal string to UIColor object. Can fail if string format is invalid.

    Following formats are accepted:

    • 0x…
    • 0X…
    • #…

    String value must be a six-digit/three-byte or eight-digit/four-byte hexadecimal number

    Declaration

    Swift

    convenience init?(hexString string: String, alpha: CGFloat? = nil)

    Parameters

    hexString

    Color value in hex format.
    alpha

    Alpha value to append to the UIColor object. Replaces existing value in hex string if present.
  • isRGBAEqual(to:)

    Checks if two colors are equal by comparing their RGBA components. This method is color space agnostic (e.g. HSV color will be treated as RGB).

    Declaration

    Swift

    func isRGBAEqual(to anotherColor: UIColor) -> Bool

    Parameters

    anotherColor

    A color to compare with.

    Return Value

    true if both colors have the same RGBA values.
